# 🤖 AutoML AI Agent (Botpress)

This project is an AI-powered AutoML agent built using Botpress.

## 🚀 Features

* Understands dataset descriptions
* Detects classification or regression problems
* Explains ML pipeline (preprocessing, training, selection)
* Generates PyCaret-based ML code
* Beginner-friendly AI assistant

## 🧠 Built With

* Botpress Autonomous Node
